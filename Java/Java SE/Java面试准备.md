## 集合概述
	也叫容器，由两个接口派生
	Collerction：存放单一元素
	Map：存放键值对
	主要子接口：List，Set，Queue

### List，Set，Queue，Map之间的区别
1. List 存储元素有序，可重复
2. Set 存储元素不可重复
3. Queue 特定排队规则确定先后顺序，存储元素有序，可重复
4. Map 使用键值对进行存储，key是无序的，不可重复的，value是无序的，可重复的，每个键最多映射到一个值
